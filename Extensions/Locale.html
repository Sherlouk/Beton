  • init(stringLiteral:)

    A convenience initializer that allows you to express a Locale using a string literal.

    let locales: [Locale] = ["en_US", "en_GB", "hu_HU"]
for locale in locales {
  print("Currency symbol: \(locale.currencySymbol ?? "N/A")")
